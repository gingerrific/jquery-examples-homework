The .toggleClass() method will take one element and allow you to pass one or more class names as a parameter. If the element currently has the class being passed, it will instead remove it (and vice-versa.) Its effects are very similar to addClass(), however with the inclusion of a .click() method you may toggle the element back to it's original class.

.hide()

The .hide() method will cause the targeted element to simultaneously move towards a height, width and opacity of 0. A duration, easing and other properties may also be provided (start, complete, etc). .show() will perform much the opposite and restore an element. The .clone() method will allow you to directly copy an element (including desendants). When used with .prepend or .append it allows for an exact copy of another element to be added to another element. The clones properties may also be altered prior to addition. .first() selects only the first element, however as this is a prepend, the added element is always the most recent. .slideUp() .delay() .toggle()

The .slideUp() method will transition the element to 0 height on trigger. This will be dependent on the elements vertical alignment and positioning. If bottom: 0, it will slide"down". The .delay() is used to delay the transition by the entered number of milliseconds. With no parameters .toggle() will simply alter the css 'display' property. It can include a transiton time element. Click "Demo" to see too separate .slideUp() and .delay() examples.

The .fadeIn() will provide a transition for the opacity of an element, however, it will also affect the elements display, even if hidden. Can be used with event triggers to animate. Click "Demo" for an example of .fadeIn()

Scroll to top
scrollTop allows a clickable link to scroll back to the top of the scrolling element by way of scrollTop: which will accept a pixel offset from the top of the scroll/coordinate to set as the top of the scroll. Animating it will scroll you to that position. Click "Demo", scroll to the bottom of the div and select "Scroll to top". Note clicking the link prior to activating the demo will retain it's default action and scroll to the top of the page.
