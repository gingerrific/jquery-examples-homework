The .toggleClass() method will take one element and allow you to pass one or more class names as a parameter. If the element currently has the class being passed, it will instead remove it (and vice-versa.) Its effects are very similar to addClass(), however with the inclusion of a .click() method you may toggle the element back to it's original class.

.hide()

The .hide() method will cause the targeted element to simultaneously move towards a height, width and opacity of 0. A duration, easing and other properties may also be provided (start, complete, etc). .show() will perform much the opposite and restore an element. Click "Demo" to start and click each circle to .hide(). Click "Demo" to .show() the elements.

The .clone() method will allow you to directly copy an element (including desendants). When used with .prepend or .append it allows for an exact copy of another element to be added to another element. The clones properties may also be altered prior to addition. .first() selects only the first element, however as this is a prepend, the added element is always the most recent. Click "Demo" to start, then select which color box you'd like to add. Double-click to reset.
